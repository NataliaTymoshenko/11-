🔹 4. UML — простою мовою

📖 Що таке UML?

Unified Modeling Language — уніфікована мова моделювання. Це міжнародний стандарт для візуального опису програмних систем.

🎯 Для чого потрібен?

  • Планувати архітектуру системи
  • Спілкуватися з командою
  • Показувати замовнику ідею
  • Знаходити помилки ще до коду
🎵

Це як:

UML для програмістів = ноти для музикантів

Загальна мова, яку розуміють усі професіонали

Стандарт Візуальний Міжнародний

🔹 6. Діаграми прецедентів (Use Case)

Актор (Actor)

Хто використовує систему?

Людина, інша програма, або навіть датчик температури!

Прецедент (Use Case)

Що система робить?

Сценарій використання — конкретна дія/функція

🌍 Переклад термінів:

Actor Актор
Use Case Прецедент
Association Зв'язок

Адміністратор

⬭ Прецеденти:

📊 Текстова діаграма:

На справжній діаграмі: актори — це stickman (людина-паличка), прецеденти — овали, а зв'язки — лінії. ![Приклад Use Case діаграми онлайн-магазину](https://www.uml-diagrams.org/examples/use-case-example-online-shopping.png)
